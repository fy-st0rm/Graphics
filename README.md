## This is my graphics concepts to learn opengl rendering

## To run the program

### - Bootstrapping build tool:
```bash
$ g++ -o cbuild cbuild.cpp
$ ./cbuild
[Usage]: ./cbuild [options]
        iso: Builds isometric example
        3d: Builds 3D example
```

### - To build isometric example:
```bash
$ ./cbuild iso
```

### - To build 3D example:
```bash
$ ./cbuild 3d
```
